# I-m-Not-Lazy
"Im Not Lazy" app — target: college adults 18–25 addicted to social media. Stack: React Native (Expo), Firebase, Sanity.io CMS, Vercel, PostHog. MVP screens: Onboarding Quiz, Setup Wizard (Instagram + YouTube first), Dashboard, Daily Check-in, Learn Feed. Content updates owned solo via Sanity. Free app.
